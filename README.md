# Die Inferenz

Deutschsprachiges AI-Nachrichtenmagazin — täglich kuratiert, recherchiert und aufbereitet mit KI-Unterstützung.

**[die-inferenz.de](https://die-inferenz.de)**

## Was ist Die Inferenz?

Ein tägliches Online-Magazin, das die wichtigsten Entwicklungen aus der Welt der künstlichen Intelligenz zusammenfasst, einordnet und auf Deutsch aufbereitet. Jede Ausgabe enthält:

- **Nachrichtenartikel** — Fakten aus mindestens zwei unabhängigen Quellen verifiziert
- **In-Depth-Reportage** — ein Thema pro Ausgabe, gründlich recherchiert
- **Tool-Radar** — neue und trendende AI-Tools des Tages
- **Aus der Werkstatt** — kuratierte YouTube-Tutorials und Trending Topics

## Wie funktioniert's?

Die Inferenz wird mit einer mehrstufigen KI-Pipeline erstellt:

1. **Aggregation** — RSS-Feeds und Web-Quellen werden automatisch gesammelt (`ai_news_agent.py`)
2. **Verifikation** — Jeder Artikel wird gegen mindestens zwei unabhängige Quellen geprüft
3. **Recherche** — Hintergrund-Recherche, Tool-Scouting, YouTube-Kuratierung
4. **Redaktion** — Einordnung, Analyse und Aufbereitung auf Deutsch
5. **Publishing** — Statische HTML-Seiten, gehostet via GitHub Pages

Mehr Details auf der [About-Seite](https://die-inferenz.de/about.html).
